这是一篇关于如何正确使用EmuAIO的全篇介绍

### 注意事项

EmuTrain系列所有软件均配有
***云控系统***
如有介意请勿使用

即 远程指令 远程信息 

### 首页
**车次查询**

一个选择器(选择动车组车次开头 C D G)

一个输入框 : 输入车次数字(1~4位)
```md
6119
```

查询按钮,点击可显示数据

可以显示 **配属路局** **配属动车所** **生产厂家** **运行交路(始发终到)** **备注** 与 该列车正在运行车次与交路时间

## 

**车号查询**

一个输入框,如:
```md
CR400AF-AZ-2311
```

一个可选按钮 显示交路信息,点击后可以显示搜索列车的交路运行

查询按钮,点击可显示数据

可以显示 **配属路局** **配属动车所** **生产厂家** **备注**

(与 可选 该列车正在运行车次与交路时间)

## 

**路局查询**

一个输入框,如:
```md
广州铁路局
```

查询按钮,点击可显示数据

可以显示 **车辆编号** **配属动车所** **生产厂家** **备注**

##

### 图鉴
上面有个选择栏,可选 **热门车型** **检测列车** **其他车型** **特殊涂装**

显示列车信息

跳转其他图鉴 , 即 [china-emu.cn](https://china-emu.cn)

##

### 关于

显示鸣谢名单与数据来源
